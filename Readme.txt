TVTestのBonDriver_UDPやBonDriver_Pipeを使ってファイル再生するプラグインです。
使い方などはTvtPlay_Readme.txtを参照してください。
バイナリはReleases欄。

◇branch:work
・更新はこのブランチに最速で適用します
・バージョン番号や説明は古いままの場合があります(利用に支障はないです)

◇branch:master
・branch:workのCommitsがある程度増えてきたらこちらにマージします
・バージョン番号を上げて説明ファイルも整理された状態です

◇branch:work-plus
・シークバーをTVTest本体のステータスバーに統合したブランチです
・branch:workやbranch:masterを不定期にマージします
